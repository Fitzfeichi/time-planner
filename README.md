# 半小时日计划

把一天拆成 48 个半小时格，在 Windows 桌面上同时记录计划、实际和状态，并用当前任务小窗把注意力拉回当下。

## 这个工具想解决什么问题

很多计划工具更适合列任务，不够适合盯执行。任务一旦写得太大，就容易停在“知道要做”，却不知道这半小时具体该做什么；如果拿手机看计划，又很容易顺手分心；纸笔虽然直接，但临时调整、拖动和回看执行偏差都不够灵活。这个工具想做的，就是把一天拆细，把任务写具体，让你更容易看清现在该做什么，以及今天到底是怎么过掉的。

## 适合谁

- 想把大任务拆进具体时间块的学生、自学者、备考用户
- 需要自己安排时间的独立工作者、自由职业者
- 想减少手机查看计划带来的走神，改用桌面工具盯执行的人
- 想同时记录“原计划”和“实际发生了什么”的人

## 为什么值得试

- 一天固定拆成 48 个半小时格，天然逼着你把任务写得更具体
- 每个时间格都能同时记录计划、实际和状态，不只知道要做什么，也知道最后做成了什么
- 当前任务卡和当前任务小窗会持续把“现在该做什么”放在眼前，减少反复切回手机的打断
- 连续任务可以合并，多选后可以拖动重排，遇到临时变化时比纸笔灵活得多
- 夜间时段可以折叠，白天的主要时间段看起来更干净
- 每天还能写今日复盘，把计划、执行和回顾串成一个闭环

## 现在已经有的功能

### 规划与记录

- 一天按 30 分钟拆成 48 个时间格，从 00:00 到 24:00
- 每个时间格都可以填写计划内容、实际内容和状态
- 支持前一天、今天、后一天切换，每个日期的数据独立保存
- 支持今日复盘，方便晚上回顾一天的安排和执行情况

### 执行与调整

- 查看当天时，会高亮当前真实时间所在的半小时格
- 右侧当前任务卡可以直接帮你定位现在这段时间要做什么
- 支持跳到当前任务，快速回到此刻所在的时间块
- 支持多选、Shift 范围选择、Ctrl切换选择
- 支持拖动重排计划内容，临时调整更顺手
- 连续时间段可以合并成一个任务块，也可以再拆分回来
- 当前任务支持“提前”和“顺延”两种调整方式，方便把上下时间段的计划快速挪动
- 夜间时段支持折叠，也可以恢复默认范围

### 专注辅助

- 支持打开当前任务小窗，单独盯住现在这一段任务
- 小窗支持置顶、最小化、关闭，也能一键回到主窗口
- 小窗里可以看到前后任务，减少“我刚刚做到哪了”的断片感
- 小窗里可以直接编辑当前任务计划，不用来回切换主界面
- 小窗会显示当前任务进度条，帮助你感知这段时间已经过去多少
- 支持单独的便利贴窗口，用来暂存临时想法，避免打断当前任务

### 桌面体验

- 这是一个 Windows 本地桌面工具，打开就能直接使用
- 主界面的侧栏宽度可以调整，并会记住你的使用习惯
- 安装版支持启动时自动检查更新
- 也支持手动检查更新，并在应用内完成下载、安装和重启

## 一个典型使用流程

### 早上

先把今天最重要的事情拆进半小时格里，不再只写“写报告”“学英语”这种太大的任务，而是写成这半小时真正要做的具体动作。

### 白天

工作或学习时主要盯着当前任务卡，或者把当前任务小窗挂在桌面角落。临时有变动时，可以直接拖动重排，也可以合并连续任务，或者用“提前 / 顺延”快速调整上下时间段。

### 晚上

回头补上实际执行情况，再写几句今日复盘。这样第二天回看时，不只是看到计划，还能知道偏差出现在哪里。

## 下载与使用

- 在 GitHub Releases 页面下载最新的 Windows 安装包
- 安装后即可直接使用
- 后续发布新版本时，可以在应用内直接检查并完成更新

## 数据说明

当前版本没有账号登录、云同步和多设备同步。计划、实际和复盘数据保存在当前电脑运行环境的 `localStorage` 中；如果你清理应用数据、浏览器数据，或者更换运行环境，这些记录可能会丢失。
